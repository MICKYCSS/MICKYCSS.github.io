使用hexo搭建博客

hexo框架，需要node.js

npm i -g hexo-cli

初始化sudo hexo init，自动化生成内容

查看生成了哪些文件ls -l

启动：hexo s[start]

新建文章：hexo n[new] ‘博客标题’

清理：hexo clean

生成：hexo g[generate]

部署博客
博客部署到github上

在github上新建一个仓库，名称为xxx.github.io
安装git部署插件
npm id —save hexo-deployer-git

打开.config.yml文件找到deploy配置
vim _config.yml
type: ’git’
repo: 仓库地址
branch: master
部署命令：hexo d[deploy]

部署完成后可以访问仓库地址

修改博客主题
如何自定义主题？
下载主题，在.config.yml中的theme修改

遇到的问题

  1. 如何添加多个ssh key
    首先，生成多个ssh key，默认的ssh key不需要重新命名
    生成的ssh key在~/.ssh文件夹下
    其中id_rsa为私钥，id_.rsa.pub为公钥
  2. 如何获取ssh key
    • 执行cat ~/.ssh/id_rsa.pub 并粘贴控制台的内容
    • 将文件拖入添加ssh key的地方
  3. 在~/.ssh文件夹下新增配置文件vim config
    遇到报错说明：已存在config文件，把这个文件删除就好,rm .config.xxx
    vim的操作
  • i为插入，退出插入按esc
  • dw为删除
  • :wq为保存
  1. config文件具体配置，HostName为git地址冒号前的内容
    如果没有配置config文件，会出现以下报错 config具体配置如下
  2. 校验是否成功的时候报错，执行命令ssh -T git@gitlab.zhiyitech.cn 解决方法：删除~/.ssh文件夹下的known_hosts文件，重新按照第一个步骤生成ssh key
  3. hexo部署到github之后报错
    解决方法：新建文件夹名称的时候报错，名称必须为用户名.github.io，同时修改项目中_config.yml文件的配置
