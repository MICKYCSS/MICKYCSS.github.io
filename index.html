Any 类型

任何类型都可以被归为any类型，本质上可以是用来逃避类型检测，由于any类型过于宽松，typescript引入了unknown类型

Unknown 类型

就像所有类型都可以赋值给any，所有类型也都可以赋值给 unknown

但是 unknown 类型的值只能赋值给 any 或者是 unknown 类型

对类型为 unknown 的值进行操作会报错

Turple 类型【元组】

Void 类型

与any类型相反，表示没有任何类型

Null 和 Undefined 类型

Never 类型

表示不存在的值的类型，表示那些会抛出异常

使用 never 避免出现新增了联合类型没有对应的实现，目的就是写出类型绝对安全的代码

typescript装饰器

装饰器是什么？

  • 是一个表达式
  • 表达式执行后返回一个函数
  • 函数的入参为 target name descriptor
  • 执行该函数后，可能配置 target 对象

属性装饰器

  • 装饰类的属性
  • 函数的入参为 target propertyKey

方法装饰器

  • 装饰类的方法
  • 函数的入参为 target propertyKey descriptor

参数装饰器

  • 装饰函数的参数
  • 函数的入参为 target propertyKey parameterIndex

  1. 如何添加多个ssh key

    其中id_rsa为私钥，id_.rsa.pub为公钥

  • i为插入，退出插入按esc
  • dw为删除
  • :wq为保存
