JavaScript精粹

Posted on 2021-12-17 Edited on 2021-12-20

第三章 对象

对象字面量

一个对象字面量就是包含在花括号中的多个键值对，如果属性名是合法的，不强制使用””包裹，如果是非合法的，则必须用””

检索

可以使用 . 或者 [] 的方法从对象中取值

更新

如果属性名已存在则更新，不存在则被扩充到对象中

引用

对象通过引用来传递，不会被复制

原型

每个对象都连接到一个圆形对象，通过字面量创建的对象连接到 Object.prototype
原型连接只有在检索值的时候才会用到，如果对象中不存在某个属性，则到原型对象中去寻找这个属性

反射

检查对象是否具有某种属性

  1. 使用 typeof 判断对象的类型
  2. 使用 hasOwnProperty 方法

枚举

使用 for in 可遍历对象中的属性，缺点是不会按照顺序遍历
如果想要按照顺序遍历，建议使用 for 循环

删除

使用 delete 删除对象中的属性，这种删除不会删除掉原型对象中的属性

第四章 函数

函数对象

函数对象连接到的原型是 Function.prototype ，函数的与众不同在于能被调用

函数字面量

函数字面量包括四个部分

  1. function 保留字
  2. 函数名称，被省略的就是匿名函数
  3. 函数入参 a 和 b
  4. 函数主体

函数调用

在调用函数的时候，除了声明时定义的形参，每个函数还接受两个附加参数 this 和 arguments，参数 this 的值取决于调用模式
调用运算符是跟在函数值表达式后面的圆括号，圆括号内包含0个或者多个表达式，用逗号隔开

方法调用模式

当一个函数被保存为一个对象的属性时，称之为方法，此时的 this 指向对象

函数调用模式

当一个函数并非一个对象的属性时，是被当做一个函数来调用，此时 this 绑定到全局对象

构造器调用模式

一个函数，如果创建的目的就是希望结合 new 前缀来调用，称之为构造函数
如果一个函数前面带上 new 来调用，那么背地里就会连接到该函数的prototype

apply调用模式

apply 接收两个参数，一个是 this 值，另一个是参数数组

arguments

函数被调用的时候，会得到一个 arguments 类数组，这个类数组里面具有参数

返回

一个函数总会有一个返回值，如果没有返回值，则返回 undefined
如果函数调用前面加上了 new 前缀，那么返回值为新创建的对象

异常

使用 try catch 捕捉 throw 抛出的错误

扩充类型的功能

递归

递归函数就是会直接或者间接调用自身的一种函数

什么是尾递归调用？

函数调用自身成为递归，如果一个函数返回自身递归调用的结果就是尾递归

作用域

JavaScript是函数作用域，意味着定义在函数中的参数和变量在函数外部是不可见的

闭包

内部函数比外部函数拥有更长的生命周期

模块

使用函数和闭包构造模块，模块模式的形式是

  1. 定义了私有变量和函数的函数
  2. 利用闭包可以访问私有变量和特权函数的特性
  3. 将特权函数返回，或者是保存

级联

给一些设置或者修改对象的某个状态却没有返回值的方法返回this，实现级联

函数柯里化

就是把函数的多个参数转换为多个单一参数的函数调用

记忆

函数可以将之前的结果存在 memo 数组中，减少不必要的运算

第五章 继承

原型

在纯粹的原型模式中摒弃类，专注于对象

函数化

继承模式的弱点是无法保护隐私，可以自定义函数返回对象

第六章 数组

长度

  1. length 没有上界 arr[1000] 也不会报错
  2. 设置更大的 length 不会分配更多空间
  3. 设置更小的 length 会把新的length属性删除
