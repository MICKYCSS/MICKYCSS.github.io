基础知识

了解Blob

  1. Blob是什么?
    Blob是表示二进制类型的大对象，它的数据可以按照文本或者二进制格式进行读取
  2. 如何创建Blob对象

了解Blob URL

Blob URL/Object URL 是伪协议，这个协议的作用就是允许 Blob 对象或者 File 对象用来作为下载的url源

url 转换为 blob对象

a标签下载方式

原理：设置 a 标签的 download 属性，结合 URL.createObjectURL 方法设置 a 标签的 href 属性c
问题：download 属性存在兼容性问题，IE11以下不适用

showSaveFilePicker API 下载

原理：利用 window 内置的 showSaveFilePicker 方法，可以选择存放图片的位置
问题：兼容性不是很好

FileSaver.js 下载

Zip 下载

原理：利用 JSZip 和 FileSaver 将文件打包后再下载
